# Guess the Number Game

This project delivers a simple "Guess the Number" console game, fulfilling the previously missing core "Game Code File" deliverable. It's a straightforward Python application designed to demonstrate basic game logic and rectify project scope misalignment.

## Setup Instructions

To run this game, you'll need Python installed on your system.

1.  **Install Python:** Ensure you have Python 3.9.x or a higher version installed. You can download it from the official Python website: [python.org](https://www.python.org/downloads/)

2.  **Clone the Repository:** If you haven't already, clone this project repository to your local machine.

3.  **Navigate to the Game Directory:**
    ```bash
    cd game_code
    ```

## Usage

Once Python is set up and you're in the `game_code` directory, you can run the game from your terminal:

```bash
python guess_the_number.py
```

Follow the on-screen prompts to play the game. The game will ask you to guess a number between 1 and 100. It will provide hints (higher or lower) until you guess correctly.
